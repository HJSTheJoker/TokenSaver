# TokenSaver v0.1.0

TokenSaver v0.1.0 is the first packaged macOS release.

## Highlights

- packaged `TokenSaver.app` for Apple Silicon Macs
- menu bar app can now be moved into `/Applications` and launched from Finder or Spotlight
- backup-first CLI installer for `distill` and `safe-distill`
- local JSONL telemetry and summary views for estimated token savings

## Install

1. Download `TokenSaver-v0.1.0-macos-arm64.zip` from the GitHub release.
2. Unzip it and drag `TokenSaver.app` into `/Applications`.
3. Launch `TokenSaver` from Applications or Spotlight.
4. If macOS blocks the first launch because the app is unsigned, Control-click the app in Finder, choose `Open`, then confirm once.

## Notes

- this first public release is unsigned and not notarized
- the app is built for Apple Silicon (`arm64`) on macOS 14+
- the CLI remains source-built in v0.1.0
